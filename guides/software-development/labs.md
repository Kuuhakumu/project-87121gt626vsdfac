## structured learning paths

pick one as your spine. too many parallel courses turns into fog.

| Platform | Best for | What u do | Cost |
|---|---|---|---|
| [The Odin Project](https://www.theodinproject.com/) | Full-stack JS or Ruby, local-first | Project-based path, code locally, push to GitHub | **Free** |
| [freeCodeCamp](https://www.freecodecamp.org/learn) | Browser-based start | ~300 challenges + projects across JS/Python/SQL/React | **Free** |
| [CS50x (Harvard)](https://cs50.harvard.edu/x/) | CS fundamentals | Lectures + problem sets (C -> Python -> SQL -> JS) | **Free** + free cert |
| [Full Stack Open](https://fullstackopen.com/en/) | Modern JS after basics | React, Node, REST, GraphQL, TypeScript, CI/CD | **Free** + free cert |
| [Boot.dev](https://www.boot.dev/) | Backend (Python -> Go) | In-browser gamified backend path | Freemium |
| [roadmap.sh](https://roadmap.sh/) | Knowing what to learn | Interactive visual roadmaps, mark progress | **Free** |

## avoid dead or stale older-guide picks

| Resource | Status | Use instead |
|---|---|---|
| **Katacoda** | shut down (2023) | [KillerCoda](https://killercoda.com/), Play with Docker |
| **CodeSchool** | gone, folded into Pluralsight | freeCodeCamp, The Odin Project |
| **Heroku free tier** | ended (2022) | [Railway](https://railway.app/), [Render](https://render.com/), [Fly.io](https://fly.io/), [Vercel](https://vercel.com/) |
| **edX free certificates** | mostly paywalled | CS50x, Full Stack Open |
| **Create React App** | deprecated by React team | [Vite](https://vitejs.dev/) |
| **Codecademy generous free tier** | heavily limited now | freeCodeCamp |
