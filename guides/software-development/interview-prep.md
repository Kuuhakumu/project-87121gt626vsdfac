## coding round - how to behave

interviewers score how u think, not just whether u get the answer.

- [ ] **Clarify** - restate the problem, ask about input size, edge cases, constraints.
- [ ] **Examples** - walk a small example by hand.
- [ ] **Approach out loud** - state the plan and Big-O before coding.
- [ ] **Code while narrating** - explain as u type.
- [ ] **Test** - walk through code with the example and edge cases.
- [ ] **Optimize** - discuss time/space improvements.

silence is the killer. a talking candidate who doesnt finish often beats a silent one who does.

## language and fundamentals questions

### JavaScript / TypeScript

- [ ] whats the difference between `let`, `const`, and `var`?
- [ ] explain `==` vs `===`.
- [ ] what is a closure? give a use case.
- [ ] what is the event loop?
- [ ] explain `async`/`await` vs callbacks vs Promises.
- [ ] what does `this` refer to, and how does it change?
- [ ] what problem does TypeScript solve? what are generics?
- [ ] explain `map`, `filter`, `reduce`.

### Python

- [ ] list vs tuple vs set vs dict - when do u use each?
- [ ] what is a list comprehension? give an example.
- [ ] mutable vs immutable types - why does it matter?
- [ ] what are `*args` and `**kwargs`?
- [ ] how does Python handle scope (LEGB)?
- [ ] what is a decorator?

### Web fundamentals

- [ ] what happens when u type a URL and press Enter?
- [ ] explain HTTP request/response and common status codes.
- [ ] GET vs POST vs PUT vs PATCH vs DELETE?
- [ ] what is REST? what makes an API RESTful?
- [ ] what is CORS and why does it block requests?
- [ ] cookies vs localStorage vs sessionStorage?
- [ ] client-side vs server-side rendering trade-offs?
- [ ] what is the DOM?

### Databases

- [ ] SQL vs NoSQL - when choose each?
- [ ] what is a JOIN? INNER vs LEFT JOIN?
- [ ] what is an index and why does it speed up queries? whats the cost?
- [ ] primary key vs foreign key?
- [ ] what is a transaction? what does ACID mean?
- [ ] how do u prevent SQL injection? parameterized queries / prepared statements.

### Git and tooling

- [ ] `git merge` vs `git rebase`?
- [ ] how do u resolve a merge conflict?
- [ ] what is a pull request and what happens during code review?
- [ ] what goes in `.gitignore` and why?

## practical / take-home

common formats:

- build a small app/API in 24-72h
- fix a buggy repo
- add a feature to an existing codebase

what they check:

- [ ] does it work and meet requirements?
- [ ] is code readable and organized?
- [ ] did u write tests?
- [ ] is there a clear README?
- [ ] does Git history tell a story?

dont over-engineer. clean, tested, working code that solves the prompt beats a sprawling half-finished framework.

## behavioral questions

answer with **S**ituation -> **T**ask -> **A**ction -> **R**esult.
prepare 5-6 stories from projects, study, work, or open source.

- [ ] tell me about yourself.
- [ ] tell me about a project youre proud of.
- [ ] describe a bug that took a long time to solve.
- [ ] tell me about a time u learned something quickly.
- [ ] describe a disagreement with a teammate.
- [ ] tell me about a time u failed or shipped something broken.
- [ ] how do u handle feedback on your code?
- [ ] why software development? why this company?

questions to ask them:

- [ ] what does a typical day look like for a junior here?
- [ ] how is code reviewed and shipped?
- [ ] what does growth/mentorship look like for juniors?
- [ ] whats the biggest challenge the team is facing?

## the AI-tools question

u may be asked how u use AI tools. the strong answer shows **judgment**, not avoidance or blind reliance.

- [ ] say u use Copilot/Cursor for boilerplate and first drafts.
- [ ] say u review every line.
- [ ] be ready with an example where AI generated subtly wrong logic or outdated APIs.
- [ ] explain any code in your portfolio.

"the AI wrote it" is a disqualifier. understanding it is the skill.

## quick-hit rapid fire

- [ ] time complexity of binary search? `O(log n)`
- [ ] hash map average lookup? `O(1)`
- [ ] LIFO data structure? stack
- [ ] FIFO data structure? queue
- [ ] difference between array and linked list?
- [ ] what is recursion? whats a base case?
- [ ] what is Big-O notation measuring?
