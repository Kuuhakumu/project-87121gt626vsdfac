## Accessibility auditing

| Lab / Tool | Platform | What you do | Cost | Status |
|---|---|---|---|---|
| [Stark](https://www.getstark.co) | Figma plugin | Run contrast checks, color-blindness simulations, and WCAG audits directly in Figma; free tier covers most needs | **Free** tier | ✅ |
| [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) | Web | Paste foreground/background hex values; instant WCAG AA/AAA pass/fail; no account | **Free** | ✅ |
| [WAVE Web Accessibility Tool](https://wave.webaim.org) | Web / browser extension | Evaluate any live page for accessibility errors; annotates in-page | **Free** | ✅ |
| [axe DevTools](https://www.deque.com/axe/devtools/) | Browser extension | Automated accessibility checks in Chrome DevTools; free extension | **Free** (core) | ✅ |
