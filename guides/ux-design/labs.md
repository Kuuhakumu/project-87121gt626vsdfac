## design tools

| Lab / Tool | Platform | What u do | Cost | Status |
|---|---|---|---|---|
| [Figma](https://www.figma.com) | Web / desktop | build frames, components, and prototypes. industry default. | **Free** (Starter) | live |
| [FigJam](https://www.figma.com/figjam/) | Web | whiteboard user flows, affinity maps, journey maps. included with Figma plans. | **Free** | live |
| [Penpot](https://penpot.app) | Web / self-host | open-source Figma alternative. cloud free forever. | **Free** | live |
| [Framer](https://www.framer.com) | Web | hi-fi interactive prototypes and real portfolio sites. | **Free** (hobby) | live |
| [ProtoPie](https://www.protopie.io) | Desktop / web | advanced sensor/logic prototyping: scroll, conditionals, multi-screen state. | **Free** watermarked; Solo ~$15/mo | live |

Adobe XD is discontinued.
dont install it, study it, or list it as current.

## accessibility auditing

| Lab / Tool | Platform | What u do | Cost | Status |
|---|---|---|---|---|
| [Stark](https://www.getstark.co) | Figma plugin | contrast checks, color-blindness simulations, and WCAG audits inside Figma. | **Free** tier | live |
| [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) | Web | paste foreground/background hex values for WCAG AA/AAA pass/fail. | **Free** | live |
| [WAVE Web Accessibility Tool](https://wave.webaim.org) | Web / browser extension | evaluate live pages for accessibility errors with in-page annotations. | **Free** | live |
| [axe DevTools](https://www.deque.com/axe/devtools/) | Browser extension | automated accessibility checks in Chrome DevTools. | **Free** core | live |
