## understand the honest career path (~1h)

**"DevOps Engineer" is almost never a first job.**
DevOps bridges development and operations, so it assumes u already know one side.

```text
Zero experience
  -> Cloud Support Associate / Cloud Ops Analyst / IT-helpdesk with cloud exposure
  -> Junior Cloud Engineer / Associate Cloud Engineer
  -> Cloud Engineer / DevOps Engineer / Platform Engineer
  -> SRE / Senior Platform Engineer / Cloud Architect
```

the guide target is your **first cloud role**: Cloud Support Associate, Cloud Ops Analyst, or Junior/Associate Cloud Engineer.

titles that usually arent realistic from zero:

- [ ] DevOps Engineer with no level
- [ ] Platform Engineer
- [ ] SRE

dont waste early applications on roles asking for production ownership u havent had yet. dw, cloud support is not a consolation prize; its the normal door in.

## mindset and expectations

- **breadth then depth.** get Linux, networking, one cloud, containers, and pipelines working, then specialize.
- **hands-on beats certs.** certs get filters; deployed, automated, observable projects get interviews.
- **automate anything u do twice.** that instinct is the job.
- **consistency beats intensity.** two focused hours a day beats a weekend binge once a month.

## pace and timeline

estimated hours: **~500-700 hours**.

| Pace | Hours/week | ~Time to job-ready |
|---|---|---|
| Casual | 5-8 | 16-22 months |
| Part-time (recommended) | 10-15 | 10-16 months |
| Full-time | 20-30 | 7-11 months |

## set up accounts (~2h)

create these now:

- [ ] **AWS Free Tier** - [aws.amazon.com/free](https://aws.amazon.com/free/). set a billing alarm immediately.
- [ ] **GitHub** - [github.com](https://github.com/). portfolio, GitHub Actions, Codespaces.
- [ ] **VS Code** - [code.visualstudio.com](https://code.visualstudio.com/) with AWS, Docker, and HashiCorp Terraform extensions.
- [ ] **optional later:** Azure free account or Google Cloud free trial.

pick **one** cloud first. AWS has the most jobs and learning material, so this guide leads with AWS. concepts transfer.
