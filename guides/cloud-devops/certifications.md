## TL;DR — What to actually get

- **Certs matter more here than in software dev.** AWS Solutions Architect Associate is a real ATS filter for cloud roles.
- **Start cheap:** AWS **Cloud Practitioner (CLF-C02, $100)** → **Solutions Architect Associate (SAA-C03, $150)**.
- **Top differentiators:** **Terraform Associate ($70)** and **CKA ($395, hands-on)**.
- **Do NOT pursue AZ-204** — retiring **July 2026**. **AZ-400** retirement is flagged. **Docker DCA is discontinued.**
- **Hands-on > cert-stacking.** A reproducible IaC project beats a 4th cert.

## Employer signal ranking

🔴 **Baseline filter:**
- **AWS SAA-C03** — the de facto entry bar for AWS roles, recognized everywhere.
- **AZ-900** — baseline for Azure-track roles; cheap, fast.
- **AWS CLF-C02** — required by many before SAA; logical first step.

🟠 **Strong differentiator:**
- **CKA** (hands-on, hardest to fake) · **Terraform Associate** (IaC fluency, $70) · **CKAD** (app-platform/SRE) · **GCP ACE** (less saturated market) · **AZ-104** (Azure beyond support).

🟡 **Nice-to-have:**
- **AWS DVA-C02** (cloud-dev roles) · **GCP CDL** (lightweight alone) · **KCNA** (K8s stepping stone) · **CompTIA Linux+/Cloud+** (gov/MSP/consulting markets).

⛔ **Skip for entry-level:**
- **AWS SOA-C02** (overlaps SAA) · **AWS DOP-C02** (professional tier, premature) · **CKS** (needs CKA first) · **AZ-204** (retiring) · **AZ-400** (flagged) · **Docker DCA** (discontinued).

## Recommended paths

**Cheapest → first job (~$320, 3–5 mo):**
`AWS CLF-C02 ($100)` → `Terraform Associate ($70)` → `AWS SAA-C03 ($150)`

**AWS-focused (~$400):**
`CLF-C02` → `SAA-C03` → `Terraform Associate` → `DVA-C02` *or* `SOA-C02` (pick by target role). Skip DOP-C02 until 1–2 yr in.

**Azure-focused (~$330):**
`AZ-900` → `AZ-104` → `Terraform Associate`. Avoid AZ-204/AZ-400 (retiring/flagged).

**Kubernetes / platform (~$645):**
`CLF-C02` → (optional `KCNA`) → `CKAD` → `CKA` (buy CKA+CKAD bundle to save). Add Terraform Associate after.
