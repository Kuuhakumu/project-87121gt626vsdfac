## TL;DR — What to actually get

- **In data, a portfolio beats certs** — but a couple of certs help get you *past the ATS filter* for analyst/BI roles.
- **Start cheap:** the **Google Data Analytics Professional Certificate** (Coursera, financial aid available) builds the SQL/viz base and is a recognized first-job signal.
- **Baseline filter for BI/analyst roles:** **PL-300 (Power BI)** — Microsoft's BI tool dominates enterprise, renews free annually.
- **Top differentiator:** **dbt Analytics Engineering Certification ($200)** — dbt is now table-stakes in the modern data stack.
- **Do NOT pursue** retired exams: **DP-203**, **DP-100** (both retired 2025), **AWS MLS-C01** (retired March 2026).
- **Hands-on > cert-stacking.** A documented end-to-end project beats a fourth cert every time.

## Employer signal ranking

🔴 **Baseline filter:**
- **PL-300** — the de facto bar for BI/analyst roles in Microsoft-heavy enterprises.
- **Google Data Analytics Cert** — recognized first-job signal for career switchers; pairs with a portfolio.

🟠 **Strong differentiator:**
- **dbt Analytics Engineering Cert** (the clearest AE signal) · **SnowPro Core** (Snowflake shops) · **AWS DEA-C01** (data engineering) · **DP-600 Fabric** (Microsoft data platform) · **Databricks DE Associate** (Databricks shops) · **AWS MLA-C01** (ML track).

🟡 **Nice-to-have:**
- **Google Advanced Data Analytics** · **IBM Data Analyst/Data Science** · **AWS AIF-C01** (lightweight GenAI literacy) · **Databricks ML Associate** · **DP-700 Fabric** · **Tableau Desktop Specialist** (Tableau shops).

⛔ **Skip / avoid (retired or premature):**
- **DP-203** (retired 2025) · **DP-100** (retired 2025) · **AWS MLS-C01** (retired March 2026) · any professional-tier ML/DE cert before you have a job in the field.

## Recommended paths

**Cheapest → first analyst job (~$165 + Coursera, 3–5 mo):**
`Google Data Analytics Cert (Coursera, aid)` → `PL-300 ($165)` → build a portfolio.

**Analytics Engineer (~$365):**
`Google DA Cert` → `dbt Analytics Engineering ($200)` → `SnowPro Core ($175)` if Snowflake-targeted.

**Data Engineer (~$350):**
`PL-300 or Google DA` → `AWS DEA-C01 ($150)` *or* `Databricks DE Associate ($200)` → add dbt cert.

**ML / Data Science long game (~$250):**
`Google Advanced Data Analytics` → `AWS AIF-C01 ($100)` → `AWS MLA-C01 ($150)` *after* you've entered as an analyst. Avoid the retired MLS-C01.
