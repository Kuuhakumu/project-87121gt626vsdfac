## what to actually get

- **portfolio beats certs** in data, but a couple certs can help with ATS filters.
- **start cheap:** Google Data Analytics Professional Certificate, with Coursera financial aid if needed.
- **baseline BI/analyst filter:** **PL-300** for Power BI.
- **top differentiator:** **dbt Analytics Engineering Certification ($200)**.
- **do not pursue:** **DP-203**, **DP-100**, **AWS MLS-C01**.
- **hands-on beats cert-stacking.** an end-to-end project beats a fourth cert.

## recommended paths

- [ ] **first analyst job:** Google Data Analytics Cert -> PL-300 -> portfolio
- [ ] **Analytics Engineer:** Google DA -> dbt Analytics Engineering -> SnowPro Core if Snowflake-targeted
- [ ] **Data Engineer:** PL-300 or Google DA -> AWS DEA-C01 or Databricks DE Associate -> dbt cert
- [ ] **ML / Data Science long game:** Google Advanced Data Analytics -> AWS AIF-C01 -> AWS MLA-C01 after entering as analyst
