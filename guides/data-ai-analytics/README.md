> this guide gets u hired into that first data role, then sets u up to grow.

## how to track your progress meow

oki so u might notice all the `- [ ]` checkboxes in this guide... heres the thing: **these boxes are display-only in this public repo** - u cant actually tick them here.

**so use your own copy:**
- **fork this repo** and tick boxes in your fork, or
- **copy the checklist into Obsidian/OneNote/Notion**, or
- **download/print it** if paper works better.

the real progress signal is the gate: the query runs, the dashboard answers a question, the project has a conclusion, and u can explain what the data can and cant prove.

## how the hour system works

timelines are in **study hours**, not weeks.

| Your pace | 680 hours takes |
|---|---|
| 1 hr/day | ~22 months |
| 2 hrs/day | ~11 months |
| 4 hrs/day | ~6 months |
| 6 hrs/day | ~4 months |

each goal is a budget, not a deadline. at **2 h/day**, this is roughly **11 months**.

## certification ladder

```text
[Foundation]      Google Data Analytics Professional Certificate
[Baseline]        PL-300 Power BI
[Differentiator]  dbt Analytics Engineering Cert + one cloud/warehouse cert
[Later]           AWS DEA-C01 / MLA-C01 / Databricks
```

> **do not pursue DP-203 or DP-100** (retired 2025) or **AWS MLS-C01** (retired March 2026). see [certifications.md](certifications.md).

## what makes this guide different

- **honest entry door** - Analyst/Analytics Engineer first, Data Scientist later.
- **SQL-first** - every track needs it and interviews screen for it.
- **hour-based** - fits any schedule.
- **verified June 2026** - cert codes, retirements, and tool versions checked.
- **region-agnostic** - no salary tables, no local job-board lists.
- **free-first** - SQLBolt, Kaggle Learn, DuckDB, BigQuery sandbox, Colab, dbt free tier.
- **current stack** - Python 3.13, pandas 3.0, DuckDB, Polars, dbt 1.11, PyTorch 2.12, with honest GenAI expectations.
