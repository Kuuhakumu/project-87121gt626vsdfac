## books worth owning

| Book | For |
|---|---|
| *Storytelling with Data* | communicating with charts |
| *SQL for Data Analysis* | analytical SQL patterns |
| *Python for Data Analysis* | pandas reference |
| *Fundamentals of Data Engineering* | DE mental model |
| *Designing Data-Intensive Applications* | deeper systems |
| *The Analytics Engineering with dbt* / dbt docs | AE track |
| *Hands-On Machine Learning* | DS/ML track |

u dont need all of these. pick the one that fits your track plus *Storytelling with Data*.
