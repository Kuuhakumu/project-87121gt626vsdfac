## How to use these (don't drown)

1. **Professor Messer + a book** for A+ — that's enough to pass.
2. **Build the home lab** (see [labs.md](labs.md)) — apply everything you watch.
3. **Jeremy's IT Lab** if you go the CCNA/networking route.
4. **M365 dev tenant** for the cloud/identity route.
5. One channel + one book + one lab per topic. More than that is procrastination.
