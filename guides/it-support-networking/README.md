> **this is the most accessible true entry point in IT.**
> unlike DevOps, Data Science, or Security, IT Support hires from genuine zero.
> the honest ladder is **Help Desk (T1) -> Desktop Support / NOC -> Junior Sysadmin / Network Tech -> Network Admin / Cloud / Security pivot**.

## how to track your progress meow

the `- [ ]` checkboxes here are display-only in this public repo. u cant actually tick them here.

use your own copy:
- **fork this repo**, or
- **copy the checklist into your notes**, or
- **download/print it**.

the real progress signal is the gate: the lab boots, the subnet math works, the ticket note is clear, and u can explain your troubleshooting process.

## certification ladder

```text
[Foundation]      CompTIA A+ (Core 1 V15 + Core 2)
[Baseline]        CompTIA Network+ (N10-009 / V9)
[Differentiator]  AZ-900 + MD-102 or CCNA
[Later]           Security+ / CCNA / AWS by pivot target
```

> **MS-900 alone is thin** and endpoint admin is now **MD-102**. see [certifications.md](certifications.md).
