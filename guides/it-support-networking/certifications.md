## what to actually get

- **certs matter a lot here.** CompTIA A+ is a genuine help-desk filter.
- **start with A+**, then **Network+** for networking fundamentals.
- **then branch:** **AZ-900 + MD-102** for cloud/endpoint/identity, or **CCNA** for networking depth.
- **MS-900 alone is thin.** modern desktop endpoint admin is **MD-102**.
- **home lab + A+ beats A+ alone.**

## recommended paths

- [ ] **first help-desk job:** `A+ Core 1 + Core 2` -> land T1 role
- [ ] **support + cloud/identity:** `A+` -> `Network+` -> `AZ-900` -> `MD-102`
- [ ] **networking:** `A+` -> `Network+` -> `CCNA`
- [ ] **security pivot:** `A+` -> `Network+` -> `Security+`

study free with Professor Messer, but verify exact current V15/N10-009 playlist before using it as the primary plan.
