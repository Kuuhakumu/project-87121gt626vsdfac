## TL;DR — What to actually get

- **Certs matter a LOT here** — more than in most other IT fields. CompTIA A+ is a genuine help-desk hiring filter; many postings list it as required.
- **Start with A+:** it's *the* entry credential. Then **Network+** for networking fundamentals.
- **Then branch by goal:** **AZ-900 + MD-102** (cloud/endpoint/identity) OR **CCNA** (networking depth).
- **MS-900 alone is thin.** Modern-desktop endpoint admin is now **MD-102** (the old MD-100/MD-101 pair was consolidated).
- **Home lab + A+ beats A+ alone** — the cert gets you past the screen, the lab gets you the offer.

## Employer signal ranking

🔴 **Baseline filter:**
- **CompTIA A+** — the de facto help-desk entry bar. Many T1 postings list it as required or strongly preferred.
- **CompTIA Network+** — expected once you target anything network-adjacent.

🟠 **Strong differentiator:**
- **CCNA** (networking depth — gold standard, opens network-tech roles) · **AZ-900** (cloud literacy, cheap, fast) · **MD-102** (endpoint/Intune — modern desktop admin) · **Security+** (security pivot, also a common gov/contractor requirement).

🟡 **Nice-to-have:**
- **MS-900** (M365 literacy, thin alone — pair with MD-102) · **ITIL 4 Foundation** (process/ticketing vocabulary, helps at enterprise/ITSM shops) · **Linux Essentials / LPIC-1** (if pivoting toward Linux/cloud) · **CCST** (Cisco's pre-CCNA entry cert).

⛔ **Skip for entry-level:**
- **CompTIA Server+, CASP+, CySA+** (premature) · **CCNP** (professional tier, years away) · stacking 4+ fundamentals certs instead of getting hired.

## Recommended paths

**Cheapest → first help-desk job (~$506, 2–4 mo):**
`CompTIA A+ (Core 1 + Core 2)` → land a T1 role. Study free with **Professor Messer** and pay only for the exams.

**Support + cloud/identity track (~$870):**
`A+` → `Network+` → `AZ-900` → `MD-102`. Best for help desk → MSP / M365 admin / endpoint roles.

**Networking track (~$975):**
`A+` → `Network+` → `CCNA`. Best for help desk → NOC → network technician → network engineer.

**Security pivot (~$900):**
`A+` → `Network+` → `Security+`. Best for support → SOC/security analyst (see the [Cybersecurity guide](../cybersecurity/)).
