## honest career ladder

IT Support is the **most accessible true entry point in IT**. u dont need a degree or prior tech job. u need A+ knowledge, basic networking, calm troubleshooting, and clear communication.

```text
Help Desk / Service Desk (Tier 1)
  -> Desktop Support / NOC Technician / Tier 2
  -> Junior Sysadmin / Junior Network Tech
  -> Network Admin / Sysadmin / Cloud Engineer / Security Analyst
```

Help Desk is not a dead end. its the launchpad. its where u learn how real environments break.

## mindset

- **soft skills are half the job.** frustrated users need calm, clear help.
- **troubleshooting is a process.** reproduce -> isolate -> test one change -> verify -> document.
- **document everything.** clear notes are how u get promoted.
- **AI deflects simple tickets.** position above pure ticket-handling: escalations, scripting, cloud/identity.
