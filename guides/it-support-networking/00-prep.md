## The honest career ladder (read this first)

IT Support is the **most accessible true entry point in all of IT.** You don't need a degree, and you don't need a prior tech job. What you need: A+ knowledge, basic networking, customer communication, and the willingness to troubleshoot calmly.

```
Help Desk / Service Desk (Tier 1)
   (6–18 mo: tickets, password resets, software installs, triage)
        ↓
Desktop Support / NOC Technician / Tier 2
   (1–2 yr: hardware, imaging, escalations, monitoring, runbooks)
        ↓
Junior Sysadmin / Junior Network Tech
   (1–3 yr: AD/Entra, patching, switch configs, automation)
        ↓
Network Admin · Sysadmin · Cloud Engineer · Security Analyst  (pivot point)
```

**Help Desk is not a dead end — it's the launchpad.** Almost everyone in cloud, networking, and security started on a help desk. It's where you learn how real environments break.

---

## Mindset

- **Soft skills are half the job.** A calm, clear, patient manner with a frustrated user matters as much as technical skill at Tier 1.
- **Troubleshooting is a process, not magic.** Reproduce → isolate → test one change → verify. Learn the method, not just the fixes.
- **Document everything.** The habit of writing clear notes is what gets you promoted off the phones.
- **AI is deflecting simple tickets.** Password resets and "how do I" questions are increasingly self-service. Position yourself *above* pure ticket-handling from day one — own escalations, learn scripting, aim at cloud/identity.
