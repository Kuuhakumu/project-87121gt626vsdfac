# Software Development — Certifications & Credentials (2026)

> Research date: 2026-06-13. Curl-verified. Confidence: ✅ / ⚠️ partial / ⛔ unverified.

## TL;DR — for SWE, portfolio > certs
For general software dev, employers weight **GitHub + deployed projects + interview performance** far above certs. Certs are **skip** for most juniors. Narrow exceptions below (cloud-adjacent, enterprise, gov).

## Credential signal ranking (junior dev role)

⛔ **Skip (no real signal for general SWE):**
- Language/framework vendor certs (Oracle Java OCP, etc.) — dated, ignored by most employers.
- "Full Stack" Coursera certificates as a hiring signal on their own.

🟡 **Nice-to-have / context-dependent:**
- **Meta Front-End / Back-End Professional Certificate** (Coursera) — ✅ exists; recognition modest; fine as structured learning, not a hiring trigger.
- **Google IT Automation with Python** (Coursera) — ⚠️ more IT-support adjacent than SWE.
- **IBM Full Stack** (Coursera) — ⚠️ existence confirmed, low employer signal.
- **freeCodeCamp certifications / The Odin Project completion** — ⛔ not a formal credential, but the *projects* built are the real value. Show the projects, not the cert.

🟠 **Differentiator (cloud-adjacent / DevOps-curious / enterprise):**
- **AWS Certified Developer – Associate (DVA-C02)** — ✅ verified, ~$150. Helps for cloud-heavy backend/DevOps roles.
- **AWS Certified Cloud Practitioner (CLF-C02)** — ✅ ~$100, entry cloud literacy.
- **Microsoft AZ-900** (Azure Fundamentals) — ✅ entry cloud, never expires.
- **GCP Associate Cloud Engineer** — ⚠️ page JS-rendered; cert active, verify code/price at cloud.google.com/learn/certification/cloud-engineer.

⛔ **RETIRING — do not start:**
- **Microsoft AZ-204** (Azure Developer Associate) — ✅ **retires July 31, 2026.** Do not pursue; await successor.

## Honest verdict
- Self-taught dev: **build projects, not certs.** A deployed full-stack app you can explain beats any entry cert.
- Pursue a cert only if: (a) targeting cloud/DevOps roles (AWS DVA-C02 or CLF-C02), (b) an enterprise/gov posting explicitly lists one, or (c) you want structured scaffolding while learning (then the Coursera pro certs are fine — but lead with the portfolio).

## What's dated (2021-era cert advice)
- Pushing Oracle Java / Microsoft language certs as hiring signals — stale.
- AZ-204 recommendations — retiring Jul 31 2026.
- Treating bootcamp/Coursera completion certificates as the credential — the projects are the credential.
