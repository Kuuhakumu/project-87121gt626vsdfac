## TL;DR - what to actually get

- **CompTIA A+ is the genuine entry filter** for help desk / desktop support. It's the one cert that opens the first door with zero experience.
- **Then Network+** (networking depth) **or AZ-900** (cloud literacy) depending on your direction.
- **CCNA** is the gold standard if you're going the networking route - high signal, but harder.
- **Cheapest strong combo to first job: A+ → Network+ (~$740 total exams)**, supplemented by free study (Professor Messer).
- ⛔ **MS-900 is being de-emphasized; MD-101/MD-100 retired → replaced by MD-102.** Don't chase retired modern-desktop exams.

## Employer signal ranking

🔴 **Baseline filter:**
- **CompTIA A+** - the de facto entry bar for help desk / desktop support, recognized everywhere.
- **Google IT Support Certificate** - increasingly accepted as a pre-A+ foundation; cheap, beginner-built.

🟠 **Strong differentiator:**
- **Network+** (networking depth) · **CCNA** (gold standard for network roles, hands-on) · **AZ-900** (cloud literacy, cheap, fast) · **MD-102** (Intune/endpoint - the modern desktop role).

🟡 **Nice-to-have:**
- **Security+** (security pivot, also DoD 8570 baseline) · **AWS CLF-C02** (AWS-shop cloud literacy) · **ITIL 4 Foundation** (process/ITSM shops, enterprise service desks) · **MS-900** (M365 literacy, lightweight).

⛔ **Skip / retired:**
- **MD-100 / MD-101** (retired → folded into **MD-102**) · chasing **MS-900** as a primary cert (lightweight; AZ-900 carries more weight) · any Windows 7/legacy-era cert.

## Recommended paths

**Cheapest → first help desk job (~$575, 2-4 mo):**
`Google IT Support Cert (~$49/mo)` → `CompTIA A+ (~$526)` → apply to help desk / MSP roles.

**Networking track (~$895):**
`A+` → `Network+` → `CCNA` (the depth combo for network tech → network admin).

**Cloud/modern-workplace track (~$790):**
`A+` → `AZ-900 ($99)` → `MD-102 ($165)` (help desk → cloud/endpoint admin via M365/Intune).

**Security pivot (~$930):**
`A+` → `Network+` → `Security+` (the classic CompTIA trifecta, opens the door to a security pivot).
